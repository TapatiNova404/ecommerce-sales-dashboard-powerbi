# 🛒 Ecommerce Sales Dashboard – Power BI

## 📌 Project Overview

This project is an interactive Ecommerce Sales Dashboard built using Power BI to analyze sales performance, customer behavior, profit trends, and payment insights.

The dashboard provides meaningful business insights through interactive visualizations and KPI tracking. It helps identify high-performing states, profitable months, preferred payment methods, and top-selling product categories.

This project demonstrates skills in data visualization, dashboard design, business intelligence, data cleaning, and DAX calculations using real-world ecommerce sales data.
